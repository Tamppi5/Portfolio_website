## Mikä tehtävässä oli haastavaa? Muuttuiko sivusto matkan varrella alkuperäisestä suunnitelmastasi?
Päädyin erilaiseen värimaisemaan kuin olin alun perin ajatellut Figma-luonnoksessa. Sain lisää inspiraatioita tutkimiltani portfolionettisivuilta figma-luonnostelun jälkeen ja totesin tämänkaltaisen suunnittelun toimivan paremmin ja olevan selkeämpi. Minusta nettisivuista tuli hienommat kuin alkuperäisestä Figma-suunnitelmasta, pyrinkin kunnolla panostamaan nettisivuihin. Tämänkaltainen suunnitelma oli varmasti saavutettava ja pidin myös kuvien sommittelusta mukaan portfolioon, sillä ne toivat eloisuutta muotoiluun. Kirjoitin myös kaikista esittelemistäni projekteista lyhyen selostuksen, mitä ei ollut vielä Figmasuunnitelmassa.

Ehkä haastavinta oli juuri CSS-muotoilujen tekeminen ja viilaaminen. Tuntuu että minulla meni hyvinkin paljon aikaa kohtuullisen merkityksettömiin viilailuihin muotoilun suhteen. Lopputulos kuitekin miellytti hyvin silmää. Jos parantaisin vielä jotain niin keskittyisin varmaan etusivun taustakuvan parempaan renderöintiin pienemmillä näyttökoilla.
